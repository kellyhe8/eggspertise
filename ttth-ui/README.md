# Talk to the Hand

This is an application for learning ASL.

- open 2 terminals; one for the front end and one for the backend
  - for the backend:
    - `cd ttth-ui`
    - `npm i`
    - `npm start`
  - for the front end:
    - `cd ttth-ui/client`
    - `npm i`
    - `npm start`
    - allow webcam access
  - Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
